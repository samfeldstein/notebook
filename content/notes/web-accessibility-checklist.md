**Local checks**:

- Add Skiplink
- Add `:focus` to `:hover` rules
- Test keyboard accessibility
- Test screen reader
- Check Firefox accessibility tab

**Once deployed:**

- [Validate HTML](https://validator.w3.org)
- [Audit](https://wave.webaim.org)
